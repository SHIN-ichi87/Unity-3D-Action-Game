# Unity 3D Action Game

![topIMG](images/18箱と空飛ぶモノ写真.png)

## 概要
キャラクターを操作して、次々と現れる敵を倒していく3Dアクションゲームです。 近接攻撃や魔法（HP/MP回復）、必殺技を駆使してワールド上に出現する敵を撃退できます。

## 制作時期
2023年（大学1年）

## 使用技術

- C#（システムロジック 約600行）

## できること・ゲームシステム
1. **キャラクター移動・カメラ操作**
2. **ステータス管理 (HP / MP)**
    - HP (体力): 敵の攻撃を受けると減少。0になるとゲームオーバー。
    - MP (魔力): 強力な攻撃や、特殊な回復魔法を使用するために必要。
3. **リソースサイクル**
    - 回復魔法でHP / MPを溜めることが可能。
    - 敵を倒すとMPが還元される（敵のMP値がプレイヤーに加算される）
4. **敵のAI・スポーンシステム**
    - 敵は常にマップ上で動いており、近づくと見つかり、プレーヤーを追いかけてくる。
    - 初期配置の敵に加え、一定時間ごとに敵が追加スポーンする。
5. **ダメージ計算**
    - プレイヤーの攻撃タイプによって「ダメージ」と「攻撃範囲」が変化する。
6. **ゲームオーバーとリスタート**
    - プレイヤーが撃破されると、ゲームオーバー画面が表示される。
    - 画面上の操作でシーンの再読み込み（リスタート）が可能。
